ğŸ¯ Quick Customization

1. Update Contact Information

Edit in src/js/config.js:

const CONFIG = {
    PHONE: '+91-YOUR-PHONE',
    EMAIL: 'your-email@example.com',
    WHATSAPP: '+91-YOUR-WHATSAPP'
};

2. Add Properties

Edit in src/data/properties.js - Add new property objects to the array.

3. Change Colors

Edit in src/css/main.css - Update CSS variables:

:root {
    --primary-color: #2d5a3d;
    --secondary-color: #d4a574;
    /* ... */
}

4. Update Contact Form Email

Currently shows toast messages. To send emails, integrate:

  • EmailJS - Simple integration
  • SendGrid API - Professional
  • Formspree - Free option
