================================================================================
                    TROUBLESHOOTING
================================================================================

PROBLEM: "Email failed to send"
SOLUTION:
  1. Check .env - SENDER_PASSWORD must have no spaces
  2. Verify it's an APP PASSWORD (not your Gmail password)
  3. Check if 2-Step Verification is enabled
  4. Restart backend servers

PROBLEM: Emails not in inbox
SOLUTION:
  1. Check spam/junk folder
  2. Verify email address is correct
  3. Check backend console for error messages
  4. Test again

PROBLEM: Backend won't start
SOLUTION:
  1. Check .env syntax (values should not have quotes)
  2. Make sure Python is installed
  3. Check if port 3001 is available
  4. Look for error messages in terminal

PROBLEM: Forms not submitting
SOLUTION:
  1. Check browser console (F12) for errors
  2. Verify backend is running (http://localhost:3001/api/health)
  3. Restart both frontend and backend
  4. Try again
